# JumpRopeCount
 Vision Based Jump Rope Counter 

A computer vision based Jump Rope Counter which detects: 
* High Knees
* Criss Cross 
* Jumping Jacks
* Normal Jumps 

The application detects the jumps with a 95% accuracy in static Background videos
